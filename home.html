Arduino powered Mac Mini Base:

These stand is designed specific for the Mac Mini 2009.It's feature an RFID login system and a usb port in front to avoid the critical design with include port only on the back.

  • The stand is entirely 3d printed and is design to increase the airflow from the bottom.

  • The RFID module is controlled by an arduino pro micro, witch can be modified or upgraded with the recognition of more (or another) CARD or integrated with some python code to automate some stuff.

  • The front present an usb port type 2; a necessary upgrade cause the machine present ports only on the back. On ther side of the usb port are 2 leds for the RFID lecture ; red wrong card , blue login granted.
